useEffect 
it is hook that is used to manage side effects (data fetching from api,
 localstorage, settimeout setinterval)

import {useEffect} from 'react'

1st - without any empty or dependenccy array
it will run whenever the state changes
useEffect(()=>{
    console.log('running')
    fetchdata()
})

2nd- useeffect with empty array or empty dependency
it will run only one time
useEffect(()=>{
     console.log('running')
},[])

3rd - useeffect with dependency
it will run only when there is a change in dependency
useEffect(()=>{
     console.log('running')
},[apidata])
